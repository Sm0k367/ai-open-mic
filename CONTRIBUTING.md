# Contributing to AI Open Mic

Thank you for considering contributing to AI Open Mic!

## How to Contribute

1. **Fork the repository** on GitHub.
2. **Create a feature branch** (`git checkout -b feature/amazing-feature`).
3. **Make your changes** following the existing code style (TypeScript, Tailwind, shadcn/ui patterns).
4. **Test thoroughly** on multiple devices (mobile, tablet, desktop) — the layout is designed to be fully responsive with `flex-col lg:flex-row` stacking.
5. **Update documentation** if you add new features (especially responsiveness or accessibility notes).
6. **Commit with clear messages** (`git commit -m 'Add amazing feature'`).
7. **Push to your branch** and open a Pull Request.

## Development Setup

```bash
git clone https://github.com/Sm0k367/ai-open-mic.git
cd ai-open-mic
npm install
npm run dev
```

Add your `GROK_API_KEY` and Stripe keys in `.env.local` for full functionality.

## Code of Conduct

Be respectful, constructive, and inclusive. We welcome contributions that improve the experience for real human creators in the AI community.

## Reporting Bugs

Open an issue with:
- Description of the bug
- Steps to reproduce
- Device/browser/screen size where it occurs
- Screenshot if possible

Thank you for helping make AI Open Mic a legitimate, high-quality platform for creative AI events!
